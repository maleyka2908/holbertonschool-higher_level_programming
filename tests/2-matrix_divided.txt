The `matrix_divided` function divides all elements of a matrix.

Usage:
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

    >>> matrix = [[1.2, 2.4], [3.6, 4.8]]
    >>> matrix_divided(matrix, 1.2)
    [[1.0, 2.0], [3.0, 4.0]]

Checking for Infinity and NaN:
    >>> import float
    >>> matrix_divided([[1, float('nan')], [3, 4]], float('inf'))
    [[0.0, nan], [0.0, 0.0]]
