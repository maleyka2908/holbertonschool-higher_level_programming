# Python - Abstract Base Classes (ABC)

This project covers advanced Python concepts including Abstract Base Classes, Duck Typing, and specialized inheritance.

## Tasks
- `task_00_abc.py`: Creating abstract classes.
- `task_01_duck_typing.py`: Implementing duck typing.
- `task_02_verboselist.py`: Extending built-in list with notifications.
- `task_03_countediterator.py`: Tracking iterator usage.
- `task_04_flyingfish.py`: Handling multiple inheritance.
- `task_05_dragon.py`: Using Mixins for complex behaviors.
