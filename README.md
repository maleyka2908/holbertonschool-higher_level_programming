# Holberton School Higher Level Programming - Python

This repository contains projects for high-level programming using Python. Each directory focuses on different concepts from basic syntax to server-side rendering.

## Projects
- **python-classes**: Object-oriented programming concepts.
- **python-everything_is_object**: Understanding mutable and immutable objects.
- **python-test_driven_development**: Writing tests before code using doctest.
- **python-server_side_rendering**: Building web applications with Flask.
- **tests**: Comprehensive test suites for various functions.
