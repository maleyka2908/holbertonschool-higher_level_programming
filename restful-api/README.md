# Restful API Project

This project focuses on building and interacting with RESTful APIs using different Python tools, from basic HTTP servers to Flask with JWT security.

## Tasks
- `task_02_requests.py`: Consuming data from external APIs using the `requests` library.
- `task_03_http_server.py`: Building a simple API using Python's built-in `http.server` module.
- `task_04_flask.py`: Developing a full-featured API using the Flask framework.
- `task_05_basic_security.py`: Implementing JWT (JSON Web Token) authentication and security for API endpoints.
