# Tests Directory

This directory contains doctest files for the Python Test-Driven Development (TDD) project.

## Files
- `0-add_integer.txt`: Tests for the `add_integer` function.
- `2-matrix_divided.txt`: Tests for the `matrix_divided` function.
- `3-say_my_name.txt`: Tests for the `say_my_name` function.
- `4-print_square.txt`: Tests for the `print_square` function.
- `5-text_indentation.txt`: Tests for the `text_indentation` function.

## How to run tests
To run the doctests, use the following command:
\`python3 -m doctest -v ./tests/<test_file_name>.txt\`
