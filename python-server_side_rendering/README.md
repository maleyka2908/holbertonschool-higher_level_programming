# Python - Server Side Rendering

This project focuses on building web applications with Flask, using Jinja2 for dynamic template rendering, and managing data from JSON and SQLite.

## Tasks
- `task_00_intro.py`: Basic setup and template rendering.
- `task_01_app.py`: Creating routes and handling dynamic content.
- `task_02_logic.py`: Implementing logic inside templates (loops, conditionals).

## Project Structure
- `/templates`: Contains HTML files for rendering.
- `/tests`: Doctest files to ensure function accuracy.
